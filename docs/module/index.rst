#########################
环境变量管理工具module
#########################

.. note:: 本章介绍如何使用环境管理工具

为了满足不同用户或者科学研究的需求，ChinaSRC-P中安装了各种版本的软件。

用户可在同一软件的不同版本之间切换，也可以在同一功能的不同软件之间切换，以此来选择最合适的编程环境和运行环境。

使用系统命令 ``module`` 可以快速地达到这一效果。

详细的介绍可以参考: [module官方网站](https://modules.readthedocs.io/en/latest/)
